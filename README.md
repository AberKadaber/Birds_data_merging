# Соединение данных первичных и вторичных отловов

## Требования к файлам
Возможно я немного поправлю и требования будут менее жесткие, но пока они следующие:
- Файл с первичными отловами должен называться `first.csv`. Разделитель должен быть ";"
- Файл с вторичными отловами должен называться `second.csv`. Разделитель должен быть ";"
- В обоих файлах должны содержаться три колонки с названиями `Ring`, `Date`, `Time` 
  (т.е. с большой буквы и без лишних пробелов в конце)
- Остальные столбцы могут быть с любыми названиями, главное чтобы они совпадали в первом и втором файле
- Порядок столбцов в файлах должен быть одинаковым (неважно каким, главное одинаковым)
- Числа с плавающей точкой (дроби) должны быть отделены точкой, а не запятой
- Формат даты либо `dd:mm:yyyy` (например 30.04.2024) либо `dd:mm::YY` (например 30.04.24)
## Запуск

### Копирование кода на свой компьютер
Чтобы запустить данную утилиту на своих файлах, сначала скопируйте этот
репозиторий себе на компьютер

Для этого есть два способа:
-  Установить гит на компьютер и вызывать следующую команду из папки, в которую вы хотите скачать проект

   `git clone https://github.com/AberKadaber/Birds_data_merging.git`

    В таком случае у вас будет возможность автоматически забирать обновления данной
    утилиты при помощи консольной команды `git pull` (или скрипта `update_repo.bat`)
   
    Чтобы открыть папку в консоли, откройте её в проводнике, нажмите правой кнопкой мыши и выберите: `Открыть в терминале`
- Скачать данные репозиторий как `.zip` архив (все в той же вкладке `Code` самая нижняя строчка). В таком случае, если я обновлю
  данную утилиту вам придется заново скачивать архив и класть в папку все нужные файлы

### Установка всех зависимостей
Утилита использует внешние библиотеки, их надо предварительно установить. Для этого откройте 
консоль и пропишите туда команду `pip install requirements.txt` (или запустите файл 
`install_requirements.bat`)

### Запуск программы
- Для объединения двух файлов в один используйте консольную команду `python merge.py` 
  (или скрипт `merge.bat`)
- Для валидации данных по птицам используйте консольную команду `python validate.py`
  (или скрипт `validate.bat`)


## Результаты
В файле `out.cvs` окажутся результаты объединения двух файлов

В файле `errors.txt` окажутся все записи с ошибками валидации птиц с подробным описанием ошибки
